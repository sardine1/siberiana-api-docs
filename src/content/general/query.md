## Offset

Параметр запроса Offset используется для исключения из ответа первых <i>N</i> элементов коллекции ресурсов.

### Пример

Для того, чтобы показать в ответе все объекты, начиная с 5:

#### Request

```
GET /collection/v1/objects?Offset=4
```
